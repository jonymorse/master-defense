Gesture Control Launcher

Select an option to launch the presentation with gesture control

Demo Mode

Launch a simple test page to try the gesture control system

Useful for testing your webcam and practicing gestures

Presenter Mode

Launch the presentation with gesture control and presenter tools

Includes controls and the ability to open a synchronized viewer window

Viewer Mode

Clean view for audience without controls or webcam

Ideal for projecting to an audience or second screen

Full Presentation

Launch the full LSU presentation with basic gesture control

Standard single-view mode with gesture navigation

Troubleshooting

Test your webcam and diagnose gesture control issues

Use this if you're having problems with the gesture control

⚠️ Webcam Access Required

Both options will request access to your webcam for the gesture control feature to work.

You can control the presentation with these gestures:

  • Swipe Right: Move to the next slide
  • Swipe Left: Move to the previous slide
